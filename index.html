WELCOME

Welcome to SubFlow

SubFlow is a professional subscription management dashboard designed to help organizations monitor, audit, and optimize all recurring software and service expenses in real-time. Built as part of the 23IT721 Full Stack Development Laboratory, this application demonstrates a complete, deployment-ready web interface.

Our Vision

To eliminate wasted SaaS spend globally, enabling teams and businesses to gain absolute clarity, control, and efficiency over their recurring digital investments.

Our Mission

Delivering an intuitive, smart tracking system that audits subscriptions, automates invoicing, sends renewal reminders, and highlights cost-saving opportunities.

ABOUT

About SubFlow

SubFlow was developed as a capstone project for the Full Stack Development Laboratory course (23IT721). It showcases modern HTML5 semantic structure, responsive CSS layout, and JavaScript-driven interactivity — all deployed on Vercel with version control via GitHub. The platform targets IT managers, finance teams, and startup founders who need a single source of truth for all subscription-based expenditures.

FEATURES

Key Platform Features

Industry-leading features to handle subscription models of all scales:

  • Automated Renewal Warnings: Get notified via email 7 days before any license renews.
  • Multi-Currency Support: Convert all global subscriptions into a single currency for unified budgeting.
  • Granular Spending Analytics: Break down costs by department, user, category, or payment source.
  • License Allocation Audit: Identify inactive accounts and empty software seats automatically.
  • Role-Based Access Control: Assign admin, manager, or viewer roles to team members securely.

ARCHITECTURE

Application Modules

SubFlow operates using five highly optimized backend and frontend modules:

Module ID Module Name Description Status
MOD-001 Identity & Access Manager Manages role-based access control, secure OAuth authorization, and organization profiles. Active
MOD-002 Sync Integration Gateway Fetches real-time transaction updates from bank API feeds, Stripe, PayPal, and vendors. Active
MOD-003 Cost Forecasting Engine Calculates future subscription expenses based on trends, renewals, and planned upgrades. Active
MOD-004 Notification Dispatcher Automates alerts across email, SMS, and webhook channels for upcoming invoices and renewals. Active
MOD-005 Audit Optimization Bot Monitors software logins to detect and flag unused user licenses and duplicate subscriptions. Testing
